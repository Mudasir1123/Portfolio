## ✨ Overview

This is a premium, high-performance portfolio website built for **Muhammad Mudasir**. It features a modern, ultra-responsive design with cinematic 3D elements and smooth animations, designed to showcase a professional journey in software engineering and AI.

### 🌟 Key Features

- 🌌 **Interactive 3D Particles**: Immersive 3D background effects powered by Three.js and React Three Fiber.
- 🎭 **Smooth Animations**: Seamless transitions and micro-interactions using GSAP and Framer Motion.
- 📜 **Lenis Smooth Scroll**: Institutional-grade scrolling experience.
- 📧 **Server-side Contact Form**: Professional communication channel integrated with Nodemailer & Gmail SMTP.
- 📱 **Fully Responsive**: Optimized for all devices, from mobile to ultra-wide displays.
- ⚡ **Next.js 15+ & React 19**: Utilizing the latest bleeding-edge web technologies for maximum performance.

## 🛠️ Tech Stack

### Frontend & UI
- **Core**: Next.js 15 (App Router), React 19, TypeScript
- **Styling**: Tailwind CSS 4.0, CSS-in-JS (Radix UI)
- **Animations**: GSAP, Framer Motion, Lenis Smooth Scroll
- **Components**: Radix UI (Accordion, Dialog, Tabs, etc.), Lucide Icons

### 3D & Graphics
- **Engine**: Three.js
- **React Bindings**: @react-three/fiber, @react-three/drei

### Backend & DevOps
- **Email Service**: Nodemailer (SMTP Integration)
- **Deployment**: Vercel
- **Analytics**: Vercel Analytics

## 🏁 Getting Started

### 1. Prerequisites
Ensure you have [Node.js](https://nodejs.org/) installed (v18+ recommended).

### 2. Installation
```bash
git clone https://github.com/Mudasir1123/portfolio.git
cd portfolio
npm install
```

### 3. Environment Setup
Create a `.env.local` file in the root directory and add your SMTP credentials:
```env
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=your-email@gmail.com
SMTP_PASS=your-app-password
CONTACT_EMAIL=your-email@gmail.com
```

### 4. Running Locally
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) to view the application.
